## ✨ Features

- 🎨 **Modern Dark Theme** - Cyan, Blue, and Slate color scheme
- 📱 **Fully Responsive** - Looks great on all devices
- 🌟 **Smooth Animations** - Loading screen, typing effect, scroll animations
- 🎯 **Interactive Elements** - Custom cursor, skill progress bars, hover effects
- ⚡ **Fast & Lightweight** - No heavy frameworks, pure vanilla JS
- 🔄 **Auto-updating Stats** - Animated counters on scroll

## 🚀 How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/afllaabdi/portofolio.git
   ```

2. **Open in browser**
   ```bash
   # Open index.html directly or use a local server
   npx serve .
   ```

3. **Customize**
   - Edit `index.html` to change content
   - Modify `styles.css` to change colors/design
   - Update `script.js` for new features
